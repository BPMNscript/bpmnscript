# External Libraries

This document lists all third-party libraries used in BPMNscript.

| Library | URL | License | Usage |
|---------|-----|---------|-------|
| Langium | <https://langium.org/> | MIT | Language workbench: grammar, parser, AST, LSP server |
| bpmn-moddle | <https://github.com/bpmn-io/bpmn-moddle> | MIT | BPMN 2.0 XML reading, writing, and programmatic construction |
| Vitest | <https://vitest.dev/> | MIT | Test framework |
| ESLint | <https://eslint.org/> | MIT | Linting |
