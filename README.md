# BPMNscript

A textual domain-specific language for authoring BPMN 2.0 processes, with first-class IDE support via VS Code.

> **Status:** Under active development as part of a Bachelor thesis. Not yet ready for use.

## Overview

BPMNscript provides a concise, developer-friendly textual syntax for BPMN process modeling. It compiles to standard BPMN 2.0 XML and targets the [Operaton](https://operaton.org/) execution engine.
